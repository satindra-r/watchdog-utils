# Watchdog-Utils
> Library for user management for scout and watchdog

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/sdslabs/watchdog/blob/master/LICENSE.md)

Watchdog-Utils is a rust library for Watchdog and Scout that controls user and role management from a keyhouse repository.

## Contents

* [Usage](#usage)
* [Development](#development)
* [Contact](#contact)

## Usage

Initialise a tokio runtime and call the `process_update_request` function.

## Development

You need to have [Rust](https://www.rust-lang.org/tools/install) installed

Open your favourite terminal and perform the following tasks:-

1. Clone this repository.

```bash
$ git clone https://github.com/satindra-r/watchdog-utils
```

2. Make the required changes inside the source code directory ([src/](src/))

3. Run `cargo test` to test your changes.

4. Rebuild the binary using `cargo build` command.
